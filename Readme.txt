Juego basico.

Se genera cartas mientras el jugador quiera.

Reglas:

Si la suma de las cartas generadas es  mayor de 7,5 se acaba el juego.
Si no quiere mas cartas se acaba el juego.
Si las cartas son 10,11,12 el valor sera 0,5 si nos sera el mismo valor que la carta generada
Si la carta es 8 o 9 no sirve, debe generar otra estas  no juegan.
